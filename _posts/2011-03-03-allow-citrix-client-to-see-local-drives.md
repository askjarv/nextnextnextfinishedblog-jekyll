---
title: "Allow Citrix client to see local drives"
date: 2011-03-03
---

I have a remote citrix user who, upon logging into citrix remotely today, has clicked “NO” to the security prompt asking user if Citrix is allowed to access their machine (upon log-in to citrix), and they now need to transfer data to their laptop’s local hard drive but obviously can’t. He can still see his local client disk within citrix, just not able to access and has been able to access it before from that laptop.

Getting user to log off citrix, reboot the laptop and log back into citrix did not force the prompt to show up for user to click YES to the message- not as easy as you’d hope! The fix for this is here:

[http://support.citrix.com/article/CTX568194](http://support.citrix.com/article/CTX568194)
